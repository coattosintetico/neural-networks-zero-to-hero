# Neural networks: zero to hero

Personal repo in which I store jupyter notebooks to work alongside Karpathy's [nn-hero-to-zero course](https://github.com/karpathy/nn-zero-to-hero/tree/master). They serve as notes for myself.

## Jupyter notebooks and uv

I learnt how to connect a jupyter notebook with a uv project with [their official docs](https://docs.astral.sh/uv/guides/integration/jupyter/#using-jupyter-as-a-standalone-tool), specially the [Using Jupyter from VS Code section](https://docs.astral.sh/uv/guides/integration/jupyter/#using-jupyter-from-vs-code).
